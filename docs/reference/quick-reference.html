Common Patterns ​

Mark Notes for Review ​

Rule 1:
  Condition: status = "draft" AND tags exists
  Action: SET status "ready-for-review"

Rule 2:
  Condition: status = "draft" AND tags exists
  Action: SET reviewed_date "2025-11-24T22:10:00"

See full example

Archive Old Notes ​

Condition: created_date < "2024-01-01"
Action: ADD tags "archived", archived_date = TODAY()

See full example

Tag Migration ​

Condition: tags contains "old-tag"
Action: REMOVE tags "old-tag", ADD tags "new-tag"

See full example

Priority Flagging ​

Condition: priority > 7 AND NOT tags contains "urgent"
Action: ADD tags "urgent"

See full example

Overdue Tasks ​

Condition: HAS deadline AND deadline < TODAY() AND status != "completed"
Action: SET status "overdue"

See full example
