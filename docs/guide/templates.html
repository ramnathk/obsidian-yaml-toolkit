On this page

Templates ​

Use templates for consistent YAML structure across notes.

Template Fields ​

Define reusable field sets:

yaml
status = "draft", created_date = NOW(), tags = ["new"], author = "default"

Common Templates ​

Note Template ​

yaml
type = "note", status = "draft", created = TODAY()

Project Template ​

yaml
type = "project", status = "planning", tags = ["project"], priority = 3

Task Template ​

yaml
type = "task", status = "todo", created = NOW(), completed = false

Best Practices ​

  1. Define templates as saved rules
  2. Apply consistently across similar notes
  3. Use functions (NOW(), TODAY()) for dynamic values

See Examples for template patterns.
